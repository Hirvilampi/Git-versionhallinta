# Kurssi: Git-versionhallinta - SOF013AS2A-3002

Tekijä: Timo Lampinen

## Sisältö
Tässä repositoriossa ovat kurssin harjoitukset 1–7 (Hello-ohjelma).

### Haarat 
- master 
- develop
- tyylit

*master* -on päähaara, joka oli alunperin main. Tein kuitenkin master nimisen haaran, muutin sen päähaaraksi ja poistin main haaran. Main tuhottiin harjoituksen 5 aikana. Master on ajantasalla oleva haara.


*develop* -haara tehtiin harjoituksessa 6. Missä tehtiin kello html sivulle.

*tyylit* -haara tehtiin jossain alkuvaiheessa. En enää muista missä. Samat perustiedostot, eli varmaan liittyi Paikallinen Git harjoituksiin.

### Tiedostot

master -haaran tiedostot ovat ajantasaiset. 

| Komento | Kuvaus |
| --------| ------ |
| .gitignore | sisältää tiedostot mitä paikallisesti ei ladata githubiin |  
| README.md | on tämä tiedosto |  
| hello.html | perusharjoituksen (harjoitus 1) tiedosto |  
| index.css | tiedosto, mihin laitoin button.css, kunnes lopussa tajusin, että on järkevämpää käyttää styles.css tiedostoa, mihin on jo laitettu kaikki muut muotoilut valmiiksi harjoitukseen 6. |
| lehma.txt | git vaiheessa omissa kokeiluissa luotu tiedosto | 
| muutoksii.md | kopioitu tiedosto, mitä käytetty testauksessa kokeiltaessa toimintoja paikallinen git vaiheessa |
| scripts.js | harjoituksen 6 "kaverin" tekemä osio | 
| styles.css | harjoituksessa 4 aloitettu  ja harjoituksessa 6 täydennetty tyylitiedosto |

Käytännössä tarpeettomia tiedostoja ovat siis index.css lehma.txt ja muutoksii.md
