Tekijä: Timo Lampinen

## Sisältö
Tässä repositoriossa ovat kurssin harjoitukset 1–7 (Hello-ohjelma).

Haarat 
- master 
- develop
- tyylit

*master* -on päähaara, joka oli alunperin main. Tein kuitenkin master nimisen haaran, muutin sen päähaaraksi ja poistin main haaran. Main tuhottiin harjoituksen 5 aikana. Master on ajantasalla oleva haara.


*develop* -haara tehtiin harjoituksessa 6. Missä tehtiin kello html sivulle.

*tyylit* -haara tehtiin jossain alkuvaiheessa. En enää muista missä. Samat perustiedostot, eli varmaan liittyi Paikallinen Git harjoituksiin.
